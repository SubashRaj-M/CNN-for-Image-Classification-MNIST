# MNIST Digit Classification using CNN

Hello! This is my deep learning project for image classification. In this project, I build a Convolutional Neural Network (CNN) from scratch to recognize handwritten digits.

## What this project does
I use the famous MNIST dataset, which contains 28x28 grayscale images of numbers from 0 to 9. 
The goal of this notebook is to teach the computer how to look at these images and correctly guess which number it is. 

Inside the notebook, you will see:
* How I build the CNN model using Keras
* Model training and accuracy results
* A confusion matrix to show how well the model predicts
* Graphs showing the actual images of the numbers, including the ones the model guessed wrong (misclassified examples)

## Tools and Libraries I used
* **TensorFlow and Keras:** For building and training the neural network
* **NumPy:** For math and working with the image arrays
* **Matplotlib and Seaborn:** For drawing the graphs and showing the images
* **Scikit-learn:** For the classification report and confusion matrix

## How to use
Download the `cnn_mnist.ipynb` file and open it in Jupyter Notebook or JupyterLab. You can run all the cells to train the model yourself and see the final accuracy!
